# 大富豪アプリ

ブラウザだけで遊べる、CPU対戦（1人 vs CPU3人）の大富豪（大貧民）Webアプリです。
ビルド不要・依存ライブラリなしの単一HTMLファイルで動作します。

## 遊び方

1. [index.html](index.html) をブラウザで直接開くだけで起動します。
   - ローカルサーバーやインストール作業は不要です。
2. 対戦開始前の画面で、使用する特殊ルールにチェックを入れてゲームを開始します。
3. 手札から出せるカードはハイライト表示されるので、選択して「出す」、出せない場合は「パス」で進行します。
4. ゲーム終了後は続けて次のゲームを開始でき、その際は前回の順位に応じたカード交換（大富豪⇔大貧民は2枚、富豪⇔貧民は1枚）が自動で行われます。

## 主な特徴

- 単一HTMLファイル（HTML / CSS / JavaScriptすべてインライン）で、ダウンロードしてそのまま開けば動作
- 対戦前にルールを自由に組み合わせて選択できるセットアップ画面
- 出せるカードのハイライト表示・パスボタンの強調など、初心者でも遊びやすいUI
- カードを出したときのアニメーションや、特殊効果発動時のカットイン演出
- 捨て札の山をクリックして、これまで場に出たカードの履歴を閲覧可能
- 手札の並び順（強い順を左からにするか右からにするか）をプレー中でも変更可能
- 7渡し・10捨て・12ボンバーなどのカード選択ウィンドウはドラッグで移動可能（最小化ボタンもあり、手札を隠さず操作できます）
- ゲームを繰り返し遊べる「連戦モード」。前回の順位に応じたカード交換や、大富豪の初手開始プレイヤー決定（初戦はダイヤの3、2戦目以降は前回の大貧民）にも対応
- トグルスイッチで切り替えられる「ダウト大富豪モード」（詳細は下記）

## ルール一覧

対戦開始前の画面で、以下のルールを個別にON/OFFして組み合わせられます。

### 数字による特殊効果（トリガーとなる数字の小さい順）

| 数字 | ルール | 内容 |
| --- | --- | --- |
| 3 | スペ3返し | 場に出たジョーカー単騎を、スペードの3で打ち消せます |
| 4 | 4止め | 8切りを阻止する効果。8がN枚出されたとき2倍（2N枚）の4（ジョーカー代用可）を出せれば阻止して自分の番にできます（8切り有効時のみ選択可） |
| 5 | 5スキップ | 5を出すと、出した枚数分だけ次のプレイヤーの番がスキップされます |
| 7 | 7渡し | 7を出すと、出した枚数分のカードを相手に渡します（自由に相手を選ぶか、左隣に強制するかを選択可能） |
| 8 | 8切り | 8を出すと場が流れ、続けて自分の番になります |
| 9 | 9リバース | 9を出すとプレー順（手番の回る向き）が反転します |
| 10 | 10捨て | 10を出すと、出した枚数分のカードを手札から選んで捨てます |
| 11 | 11バック | J（11）を出すと、そのトリック中だけ強さの順序が反転します（強制発動 or 任意発動を選択可能） |
| 12 | 12ボンバー | Q（12）を出すと、出した枚数まで好きなランクを指定でき、全員がそのランクを手札から捨てます |
| - | 革命 | 同じランクを4枚以上出すと、強さの順序が逆転します |

### しばり・特殊ルール

| ルール | 内容 |
| --- | --- |
| スートしばり | 直前のプレイと今回のプレイが共通のスートを含むと、以降そのトリック中は該当スートを含めないと出せなくなります |
| 数しばり | 連続する数字が出されると、次は一つ上の数字しか出せません（11バックや革命中は逆方向） |
| ジョーカー | 山札にジョーカーを1〜8枚（デフォルト2枚）追加。単騎最強、組み合わせではワイルドカードとしても使えます |
| 禁止上がり | 特定条件（スペ3返し上がり／5スキップ上がり／7渡し上がり／8切り上がり／10捨て上がり／12ボンバー上がり／最強カード上がり／ジョーカーワイルド上がり）で上がると反則になり、その時点で最下位候補になります |
| 都落ち | 前回の大富豪が今回1位になれなかった場合、その時点で最下位候補になります（連戦モードのみ意味を持ちます） |

## ダウト大富豪モード

セットアップ画面上部のトグルスイッチでON/OFFできる特殊モードです。ONにすると配色も一段ディープなものに切り替わります。

- カードは**出すカードを宣言（ランク＋スートまで）したうえで、すべて裏向き**に出します。宣言は実際に持っていないカードでも、既に使用済みのカードでも自由につけられます（＝ウソをつけます）。
- 自分の番の選択肢は**出す・パス・ダウト**の3つ。ダウトは**場の一番上の伏せ札**にしか宣言できません（重ねられた後は不可）。
- ダウトが外れた（宣言が本物だった）場合、失敗したプレイヤーは**アウトが1つ**たまり、そのトリックと次のトリックの間**プレイに関与できません**（7渡し・12ボンバーなど他家の効果は受けますが、5スキップの対象にはならず、休み中はダウトの宣言もできません）。
- ウソを見抜かれた場合、出したカードは**全部手元に戻り**、その時のカードの効果（革命・9リバースの反転、7渡し・10捨てで動いたカード等）も**すべて取り消し**になります。見抜いた側は**次のトリックの親**になり、見抜かれた側はアウトが1つたまり1トリック休みます。
- **アウトが3つたまると、その時点で強制敗北**（最下位候補）になります。
- 4止め・スペ3返し・（4止めなしの）8切りのように「成立すれば即座に場が流れる」効果も、ダウトモードでは他の全プレイヤーが一巡パスかダウトを選ぶまでは確定しません。
- ダウトが外れて宣言が本物だと証明されたカードは、その後捨て札に流れた時点で**表向きのまま恒久的に記録**されます（伏せたまま流れた札は中身不明のまま）。これにより「ダウト失敗を覚悟のうえで相手の手の内を確認し、二度とそのカードで嘘をつかせないようにする」という駆け引きが可能です。
- CPUの強さもこのモードに対応しており、ノーマルはほとんどダウトしません。エリート以上は公開情報から矛盾を検出してダウトし、手札にないランクの効果を安全な宣言で横取りするブラフも行います。レジェンドはさらに、そのゲーム中に暴かれたウソ・証明された正直さを学習してダウトの精度を上げ、駆け引きを仕掛けてきます。

## 技術構成

- HTML / CSS / JavaScript（フレームワーク・ビルドツールなし）
- すべて [index.html](index.html) 一つに完結しており、ブラウザで開くだけで動作します
